# BinAref Releases

Public download hub for **BinAref** apps. This repository holds release binaries
only — **no source code**. It exists so the website's download buttons and direct
links resolve for everyone, without exposing the private source repositories.

## BinAref Player

A desktop + mobile video player built with Flutter and media_kit (libmpv).
Trilingual (English / العربية / Türkçe) with full RTL support.

**Latest version: v1.3.0**

**Download** — these links always point at the newest release:

| Platform | Download |
|---|---|
| Windows — Installer | [BinAref-Player-Setup.exe](https://github.com/BinAref/BinAref-Releases/releases/latest/download/BinAref-Player-Setup.exe) |
| Windows — Portable (zip) | [BinAref-Player-Windows-x64.zip](https://github.com/BinAref/BinAref-Releases/releases/latest/download/BinAref-Player-Windows-x64.zip) |

> Windows may show a SmartScreen prompt and Android an "unknown sources" warning —
> the builds are not code-signed yet.

### What's new in 1.3.0
- **Broadcast cards** — in-app announcement cards shown at the top of the screen while you watch, with a notifications section (bell) that keeps each one.
- **Live-scrubbing seeks** — the picture moves as you drag the seek bar, drag across the video, or press-and-hold to fast-forward / rewind.
- **Floating tool dock on phones** — screenshot, lock, aspect ratio, repeat, rotate, PiP and playlist in a draggable, collapsible dock.
- **Screen stays awake** during playback; **Android screenshots** save straight to the Gallery; **branded app icon**.
